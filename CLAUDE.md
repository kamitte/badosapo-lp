## アプリ「バドサポ」の概要

### コンセプト

バドミントン大会の運営をすべてオフラインで完結できるデスクトップアプリ。
参加者の登録から組み合わせ作成、試合進行、スコアシート印刷、賞状作成まで一貫して管理できる。

### 対応OS

- Windows
- macOS

### 主要機能

1. **大会管理**: 大会の作成・編集・一覧管理
2. **種目管理**: 個人戦・団体戦の種目を柔軟に設定
3. **参加者登録**: 手動登録 + Excelファイルからの一括インポート
4. **組み合わせ（ドロー）作成**: 3つの競技形式に対応
   - トーナメント（シングルエリミネーション、3位決定戦オプションあり）
   - リーグ戦（ブロック分けに対応）
   - リーグ→トーナメント（予選リーグ→決勝トーナメントのハイブリッド形式）
5. **試合進行・スコア管理**: 21点制・3ゲームマッチのスコア入力
6. **スコアシート印刷**: 試合ごとのスコアシートをPDF出力
7. **大会結果・順位表示**: トーナメント/リーグの順位を自動計算
8. **賞状作成**: 入賞者の賞状をPDF出力

## LPの構成案

### セクション構成

1. **ヒーロー**: キャッチコピー + ダウンロードボタン
2. **特徴紹介**: 主要機能を3〜4つピックアップしてアイコン付きで紹介
3. **機能詳細**: 各機能のスクリーンショット付き紹介（スクショが用意でき次第）
4. **対応形式**: トーナメント / リーグ / リーグ→トーナメントの3形式を説明
5. **料金**: 無料で試せる旨 + プレミアムプランの案内
6. **ダウンロード**: OS別ダウンロードボタン
7. **フッター**: お問い合わせ先、利用規約など

### デザインの方向性
